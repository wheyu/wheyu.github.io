# <div align="center">md教程</div>

## 简介
- Markdown语言是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。
- Markdown语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建
- Markdown 编写的文档后缀为 .md, .markdown

## Markdown标题
1. 使用`=`和`-`标记一级和二级标题
```
我是一级标题
====
我是二级标题
------
```
我是一级标题
====
我是二级标题
------

2. 使用`#`号可表示 1-6 级标题
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
